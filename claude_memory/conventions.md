# 代码约定

## Python 后端
- FastAPI 路由放在 `backend/routers/`
- 模型/ Schema 放在 `backend/schemas/`
- 服务逻辑放在 `backend/services/`
- 配置用 `.env` + `python-dotenv`

## React 前端
- 组件放在 `frontend/src/components/`
- 页面放在 `frontend/src/pages/`
- API 调用封装在 `frontend/src/api/`
- 用 TypeScript 严格模式

## 通用
- 所有注释用中文
- 变量/函数名用英文
- commit message 用中文
